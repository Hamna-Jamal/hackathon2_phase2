# Next.js Development Skill

## Description
Next.js is a popular React-based framework for building production-ready web applications. It offers features like server-side rendering, static site generation, API routes, and file-based routing.

## Key Features
- Server-Side Rendering (SSR)
- Static Site Generation (SSG)
- Client-Side Rendering (CSR)
- File-based routing system
- API routes
- Built-in CSS and Sass support
- Image optimization
- Automatic code splitting

## Common Commands
- `npx create-next-app@latest` - Create a new Next.js app
- `npm run dev` - Start development server
- `npm run build` - Build the application for production
- `npm run start` - Start the production server

## File Structure
- `pages/` - Contains page components (for pages router)
- `app/` - Contains page components (for app router)
- `components/` - Reusable React components
- `public/` - Static assets
- `styles/` - Global stylesheets
- `api/` - API route handlers

## Best Practices
- Use dynamic imports for code splitting
- Leverage Next.js Image component for optimization
- Implement proper error handling with Error Boundary
- Use getStaticProps/getServerSideProps appropriately
- Follow accessibility guidelines
